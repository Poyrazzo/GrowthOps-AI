# Agent Instructions

**CRITICAL DIRECTIVE FOR ALL AGENTS:**

Every AI agent starting a job on this project MUST read the following files to have a good grasp of the project before making any changes or plans:

1. `current_state.md`
2. `experience.md`
3. `software_requirements_specification.md` (SRS file)
4. `architecture.md` (System Architecture)
5. The main structural parts of the codebase/system.

**CRITICAL RULE FOR ALL MODIFICATIONS:**

Whenever any changes, implementations, or bug fixes are made, the agent MUST update both `Docs/experience.md` and `Docs/current_state.md` to reflect the latest status, issues encountered, and work completed. This is mandatory to keep the project diary and single source of truth up-to-date.
